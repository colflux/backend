Relaciones

Relaciones clave

sitio es el nodo central: la mayoría de entidades lo referencian.

municipio → departamento → region N:1 en cascada
sitio → municipio, cobertura, vegetacion, disturbio, sistema_referencia N:1
parcela → sitio N:1
proyecto → sitio (sitio principal) N:1 nullable
proyecto ↔ aliado ↔ proyecto_aliado M:N
flujo_camaras → sitio N:1
torre_ec → sitio N:1
equipo / config_sensor_gas / fuente_energia → torre_ec N:1
caracterizacion_muestreo_suelo → sitio N:1
publicacion ↔ sitio ↔ publicacion_sitio M:N
publicacion ↔ autor ↔ publicacion_autor M:N
